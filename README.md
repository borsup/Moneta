## 📱 Installer l'application

Moneta est une **Progressive Web App (PWA)** — elle s'installe comme une vraie application, sans passer par un store.

### Sur iPhone / iPad (iOS)

1. Ouvrir **Safari** et aller sur [borsup.github.io/moneta](https://borsup.github.io/moneta/)
2. Appuyer sur l'icône **Partager** (carré avec une flèche ↑) en bas de Safari
3. Sélectionner **"Sur l'écran d'accueil"**
4. Appuyer **"Ajouter"** ✅

### Sur Android

1. Ouvrir **Chrome** et aller sur [borsup.github.io/moneta](https://borsup.github.io/moneta/)
2. Une bannière **"Installer Moneta"** apparaît automatiquement — appuyer **"Installer"**
3. Ou : appuyer sur les **⋮** → **"Ajouter à l'écran d'accueil"**

> 💡 Une fois installée, Moneta fonctionne **hors ligne** et ressemble à une application native.

## ✨ Fonctionnalités

- **Or** : 24K (999‰) · 22K (916‰) · 21.6K (900‰) · 18K (750‰) · 14K (585‰)
- **Argent** : 999.9 · 925 (Sterling) · 900 · 835 · 800 (Bilon)
- Poids rapides : 1 Oz · 1/2 Oz · 1/4 Oz · 1/10 Oz · 1/20 Oz
- Cours spot **en temps réel** via proxy Cloudflare
- Code couleur du résultat :
  - 🔵 **Bon marché** — prime ≤ 0%
  - 🟢 **Bon prix** — prime 0% à 8%
  - 🟠 **Prix correct** — prime 8% à 12%
  - 🔴 **Prix élevé** — prime > 12%
- Bilingue **Français / English**
- Fonctionne **hors ligne** après la première visite

---

## 🧮 Formule de calcul

```
Prime (%) = ( Prix d'achat / (Poids × Cours spot × Titre) − 1 ) × 100
```

## 📱 Install the app

Moneta is a **Progressive Web App (PWA)** — install it like a native app, no store required.

### On iPhone / iPad (iOS)

1. Open **Safari** and go to [borsup.github.io/moneta](https://borsup.github.io/moneta/)
2. Tap the **Share** icon (square with arrow ↑) at the bottom of Safari
3. Select **"Add to Home Screen"**
4. Tap **"Add"** ✅

### On Android

1. Open **Chrome** and go to [borsup.github.io/moneta](https://borsup.github.io/moneta/)
